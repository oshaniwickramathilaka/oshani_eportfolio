03

Interactive GUI for Machine Learning Models

Group Project
What it was about

Many beginners are interested in machine learning but feel discouraged by the amount of coding involved. This project aimed to address that by creating a desktop application that allows users to explore machine learning concepts visually without needing to write code.

What we built

This project focuses on developing an interactive machine learning application tailored for beginners and academic users. Unlike many existing no code tools that emphasize deployment, this platform is designed as a learn oriented system that guides users through the complete machine learning pipeline. The application is structured into multiple modules, starting with data preparation, where users can import datasets, handle missing values, and preprocess data through normalization and encoding techniques. The exploratory data analysis module enables users to visualize patterns and trends using intuitive graphs. Users can then build machine learning models using a drag and drop interface, selecting from supervised (ANN, SVM), unsupervised (K-Means), and deep learning models (CNN). The platform allows customization of hyperparameters and neural network layers, particularly for CNN architectures. A key feature of the system is real time visualizations during model training, where users can monitor metrics such as accuracy, loss, precision, and recall through live graphs. The evaluation module further supports performance analysis using confusion matrices, ROC curves, and F1-scores by enabling comparison between multiple models. Additionally, the application includes built in tutorials, contextual tooltips, and interactive visualizations to enhance learning. Users can test trained models with custom inputs. Overall, the platform serves both as a practical machine learning tool and an educational environment that helps users gradually build confidence and understanding in machine learning concepts.

My part
  • Researched regression algorithms and how to present them clearly in the app
  • Worked on improving the overall look and feel of the interface
  • Put together the project documentation and reports
What I took away
  • Gained a much deeper appreciation for how ML models actually work under the hood
  • Learned how design choices can make or break a technical tool for non technical users
  • Built something genuinely useful for learners
